# Dev Notes

## Ping + Glow Animation for Icons/Images

Used in: `src/pages/Developer.tsx` — CTA section icon

### Structure
```html
<div class="relative">
  <!-- Main icon with neon glow drop-shadow -->
  <img
    src="..."
    alt=""
    class="w-16 h-16 md:w-20 md:h-20"
    style="filter: drop-shadow(0 0 20px rgba(0,102,255,0.4));"
  />
  <!-- Duplicate absolute overlay with ping animation -->
  <img
    src="..."
    alt=""
    class="absolute inset-0 w-16 h-16 md:w-20 md:h-20 animate-ping opacity-20"
  />
</div>
```

### How it works
- **`animate-ping`** — Tailwind built-in: scales the element up while fading out, creating a radar/ripple ping effect.
- **`opacity-20`** — Makes the ping overlay faint so it doesn't overpower the main icon.
- **`absolute inset-0`** — Positions the ping overlay exactly on top of the main icon.
- **`drop-shadow(...)`** — Neon glow effect in the desired accent color (here `#0066FF`).
- **Same `w-* h-*` classes** on both elements ensures perfect alignment at all responsive breakpoints.

### To reuse in any project
Just copy the wrapper `<div class="relative">` block, change the `src`, size classes, and glow color. No custom CSS or JS required — pure Tailwind.
